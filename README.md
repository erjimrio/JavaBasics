📘 JavaBasics – Learning Exercises in Java

📌 Project Description  
This repository contains example exercises from the book *Java Programming for Beginners*.  
The main goal is to practice and reinforce fundamental concepts of the Java language while ensuring portability across multiple environments:  
- Ubuntu VM with Java 21  
- Linux MX laptop with Java 17  
- WSL2 on Windows with Java 21  

Only the `src/` folder and the `.gitignore` file are committed to GitHub, ensuring clean version control and maximum mobility between platforms.

⚠️ Attribution Notice  
Exercises are based on examples from the book *Java Programming for Beginners*.  
Content is intended solely for educational and personal practice purposes.

🧪 Technologies and Techniques Used  
- Language: Java (tested with JDK 17 and JDK 21)  
- Main IDE: IntelliJ IDEA (Windows + WSL2)  
- Version Control: Git + GitHub  
- Structure: source code in `src/`, local configs excluded via `.gitignore`  

📦 Prerequisites  
Before running the exercises, make sure you have:  
- JDK 17 or higher installed (depending on your environment)  
- IntelliJ IDEA or any compatible IDE/compiler  
- Git configured with SSH keys for GitHub  

▶️ Steps to Run the Exercises  
1. Clone the repository into your projects folder:  
   ```bash
   cd "/mnt/d/Mis Documentos/Proyectos/java"
   git clone git@github.com:usuario/JavaBasics.git
2. Open the project in your preferred IDE.
3. Configure the SDK to point to the JDK available in your environment (Java 17 or Java 21).
4. Compile and run .java files either from the IDE or using javac/java in the terminal.

🧭 Workflow
* Write exercises in src/ while studying the book.
* Test compilation across different environments (Ubuntu VM, Linux MX, WSL2).
* Commit changes with Git and push to GitHub.
* Keep .gitignore updated to exclude local configs and maintain portability.
